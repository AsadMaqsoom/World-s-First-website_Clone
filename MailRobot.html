Mail Robot

The mail robot is a program that will accept incoming mail and allow remote users to:

  • Subscribe to mailing lists (and unsubscribe)
  • Retrieve information given a W3 address (URL)

Originally from UC Berkeley, an enhanced robot is distributed as part of the world-wide web global information initiative. Further information available is:
